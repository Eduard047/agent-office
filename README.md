# Agent Office

Светлый веб-интерфейс для реальной маленькой команды AI-агентов. Оркестратор строит
короткий план, назначает роли, показывает ход работы в офисе и сохраняет последний
результат в браузере.

## Локальный запуск

1. Скопируйте `.env.example` в `.env`.
2. Добавьте `OPENAI_API_KEY` в `.env`.
3. Запустите `pnpm dev`.
4. Откройте адрес, который покажет Vite.

Ключ используется только серверным обработчиком `/api/runs` и не попадает в
браузерную сборку. По умолчанию режим «Экономно» использует `gpt-5.6-terra` для
короткого плана и `gpt-5.6-luna` для выполнения максимум трёх задач.

## Проверка

```sh
pnpm test
pnpm build
```
