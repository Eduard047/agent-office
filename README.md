# Agent Office

Тёплый визуальный офис для маленькой команды AI-агентов. Оркестратор выбирает
подходящую модель, строит короткий план, назначает роли и показывает работу
команды прямо в офисе.

## Приложение для macOS

Автономная версия работает без dev-сервера, локального проекта и API-ключа.
Внутри `.app` находятся веб-интерфейс, локальный сервер и Node.js. Приложение
поднимает их на случайном loopback-порту и запускает Codex через авторизацию
ChatGPT на этом Mac.

- Поддерживается Apple Silicon и macOS 13 или новее.
- Используются лимиты подписки ChatGPT, а не отдельный API-баланс.
- Доступны Luna, Terra и Sol, мощность от `none` до `ultra`, а также Auto.
- Вставка скриншотов работает через `⌘V`, выбор файла и перетаскивание.
- Интерфейс показывает реальный остаток окна подписки и фактическое число
  обработанных токенов.

Собрать архив:

```sh
pnpm run build:mac
```

Готовый файл появится в `release/Agent-Office-macOS-arm64.zip`.

## Как работает Auto

Маршрутизатор оценивает длину, сложность и наличие изображений локально, без
отдельного вызова модели. Простые задачи идут в Luna, обычные многошаговые — в
Terra, а сложные и требовательные к качеству — в Sol. Ultra включается только
вручную, чтобы не расходовать лимит подписки без явного решения пользователя.

К задаче можно прикрепить до четырёх PNG, JPG, WebP или GIF. Один файл может
занимать до 8 МБ, все вместе — до 24 МБ. Вложения сохраняются во временной
закрытой папке, передаются в Codex через `codex exec --image` и удаляются сразу
после завершения запуска.

## Опубликованный сайт

GitHub Pages — это витрина и ссылка на скачивание. Он намеренно не ищет
локальный сервер на компьютере пользователя: HTTPS-страница не может надёжно и
безопасно управлять процессом на Mac. Настоящие запуски через ChatGPT Pro
выполняются в автономном приложении.

Серверный API-режим проекта остаётся отдельной опцией для хостинга с
`OPENAI_API_KEY`; он не требуется приложению для macOS.
