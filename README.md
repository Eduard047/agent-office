# Agent Office

Светлый веб-интерфейс для реальной маленькой команды AI-агентов. Оркестратор строит
короткий план, назначает роли, показывает ход работы в офисе и сохраняет последний
результат в браузере.

## Локальный запуск

1. Убедитесь, что Codex авторизован: `codex login status`.
2. Если нет — выполните `codex login` и войдите через аккаунт ChatGPT.
3. Запустите `pnpm dev`.
4. Откройте адрес, который покажет Vite.

Локальная версия запускает Codex CLI через авторизацию ChatGPT и использует
лимиты вашей подписки, а не отдельный API-баланс. Можно вручную выбрать
`gpt-5.6-luna`, `gpt-5.6-terra` или `gpt-5.6-sol`, а также мощность рассуждения
от `none` до `ultra`. Режим `ultra` включается только вручную; автоматическая
маршрутизация его не выбирает, чтобы не расходовать лимит подписки без явного
решения пользователя.

В режиме `Auto` локальный маршрутизатор оценивает длину и сложность запроса без
дополнительного вызова модели. Простые задачи идут в Luna с тремя ролями,
обычные многошаговые — в Terra, а сложные и требовательные к качеству — в Sol.
Решение маршрутизатора и его причина показываются в интерфейсе.

В шапке локальной версии отображается реальный остаток окна подписки Codex в
процентах. В раскрывающейся карточке отдельно показаны время обновления лимита,
расход текущего запуска и накопленная статистика токенов. Подписка не является
фиксированным кошельком токенов, поэтому её остаток нельзя корректно показать
одним абсолютным числом.

Опубликованная серверная версия не может безопасно использовать локальную
ChatGPT-сессию с вашего Mac. Для неё API-режим остаётся опциональным.
